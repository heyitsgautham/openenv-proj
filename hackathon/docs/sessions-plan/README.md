# Sessions Tactical Plans

This folder stores tactical plan artifacts generated at the start of each session.

Naming convention:
- session-01-tactical-plan.md
- session-02-tactical-plan.md
- session-03-tactical-plan.md
- session-04-tactical-plan.md
- session-05-tactical-plan.md
- session-06-tactical-plan.md
- session-07-tactical-plan.md

Minimum sections each tactical plan should include:
1. Scope and non-scope
2. Ordered implementation steps
3. Risk and mitigation list
4. Verification checklist with pass criteria
5. Go or no-go decision
